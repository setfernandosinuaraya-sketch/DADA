Gas. Kalau yang ingin diubah hanya warna border pink/neon menjadi putih, gunakan HTML + CSS berikut.

HTML
<!-- TERMINAL SEARCH -->
<div class="terminal-search">
    <div class="terminal-title">
        🔍 <span>TERMINAL SEARCH</span>
    </div>

    <input type="text" placeholder="Find system...">
</div>
CSS
/* BOX */
.terminal-search{
    width:300px;
    padding:16px;
    background:rgba(10,10,10,.85);
    border:2px solid #ffffff;          /* Putih */
    border-radius:14px;
    box-shadow:
        0 0 8px rgba(255,255,255,.35),
        inset 0 0 15px rgba(255,255,255,.08);
}

/* Judul */
.terminal-title{
    display:flex;
    align-items:center;
    gap:8px;
    margin-bottom:12px;
    font-size:13px;
    font-weight:700;
    color:#ffffff;                     /* Putih */
    text-transform:uppercase;
    letter-spacing:1px;
}

/* Input */
.terminal-search input{
    width:100%;
    padding:10px 12px;
    background:#0b0b0b;
    border:2px solid #ffffff;          /* Putih */
    border-radius:8px;
    color:#ffffff;
    outline:none;
    transition:.3s;
}

/* Placeholder */
.terminal-search input::placeholder{
    color:#bdbdbd;
}

/* Fokus */
.terminal-search input:focus{
    border-color:#ffffff;
    box-shadow:0 0 12px rgba(255,255,255,.45);
}

Jika di script lama masih berwarna pink, cari lalu ganti:

CSS
border:2px solid #ff00ff;

menjadi

CSS
border:2px solid #ffffff;

Kemudian ganti juga:

CSS
color:#ff00ff;

menjadi

CSS
color:#ffffff;

Dan ubah efek glow:

CSS
box-shadow:0 0 20px #ff00ff;

menjadi

CSS
box-shadow:0 0 15px rgba(255,255,255,.4);

Hasilnya akan sama seperti gambar yang Anda inginkan: seluruh border dan efek neon berubah menjadi putih, sementara background tetap hitam.
